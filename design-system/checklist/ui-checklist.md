# UI Quality Checklist

Before merging or releasing, verify:

---

## Visual Consistency
- [ ] Page background is `bg-background`
- [ ] Cards use correct radius and border
- [ ] Accent color not overused

## Typography
- [ ] Correct hierarchy applied
- [ ] No bold body text
- [ ] Muted text used for secondary info

## Interaction
- [ ] All checkboxes use `ProtanniCheckbox`
- [ ] Hover actions are subtle
- [ ] Disabled states are calm

## Motion
- [ ] No bounce or elastic animations
- [ ] Durations within system range
- [ ] Stagger used on lists

## Emotional Tone
- [ ] No urgency language
- [ ] No judgmental empty states
- [ ] Calm, human wording
